# CDLA: A Chinese document layout analysis (CDLA) dataset

CDLA是一个中文文档分析数据集，面向中文文献类（论文）场景。包含10个label: Text, Title, Figure, Figure caption, Table, Table caption, Header, Footer, Reference, Equation。
